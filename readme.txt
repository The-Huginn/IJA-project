# IJA Project 2022
## Class and sequence diagram editor
### Authors: Vladimir Meciar (xmecia00), Rastislav Budinsky (xbudin05)

### This app is still in development

This project aims to deliver class and sequence diagrams editor, which will contain some basic checks for correct types, method calls in sequence diagrams and so on.</br>

The backend of the application should be already ready for release, currently GUI is being developed.</br>

The main application currently offers only working with loading already prepared diagrams and playing with types </br>
To load an diagram Navigate to File -> Open. </br>
If you would like to test adding new Type to the diagram you can click into the Add Component tab (It's a feature obv) </br>


To test correctness of the app you can try to run prepared tests in src/test
